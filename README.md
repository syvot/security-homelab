# security-homelab
Cybersecurity homelab built with Ubuntu, KVM/QEMU, and Kali Linux

## Overview
This repository documents the setup and configuration of my personal cybersecurity homelab built on Ubuntu using KVM/QEMU and Kali Linux.

The goal of this lab is to create a practical environment for learning virtualization, Linux system administration, and cybersecurity fundamentals in an isolated and reproducible way.

## Objectives
- Learn Linux-based virtualization with KVM/QEMU
- Build and manage virtual machines for security practice
- Create a foundation for future cybersecurity labs
- Document troubleshooting and performance optimizations

## Environment
- **Host OS:** Ubuntu
- **Hypervisor:** KVM/QEMU
- **VM Manager:** virt-manager
- **Guest OS:** Kali Linux
